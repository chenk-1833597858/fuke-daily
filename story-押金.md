# 短篇剧情：押金

---

## 主角的因

我放不下的是：我妈给我的那一万二，她说"好好租个房子"。现在一半被人吞了。

因为这件事，我变成了一个"该我的我必须拿回来"的人。

---

## 人物表

1. **小林（我）** — 28岁，广告公司文案，刚来上海三个月
2. **陈姐** — 45岁，二房东。五套房的包租婆
3. **陈姐老公** — 50岁，无业。比陈姐狠十倍
4. **老刘** — 50岁，楼下修车铺老板。住这条街二十年了
5. **小刘** — 隔壁租客，欠了两个月工资，比小林还穷

---

## 五大引擎定调

**钩子：** 她搬走那天，陈姐扣了她八千。说"大房东不退我，我也没钱退你。"

**弧光：** 「讲道理有用」→「讲道理没用」→「以狠对狠」→「狠完了发现回不去了」（堕落型）

**主题：** 你逼一个人走到绝路，就别怪她把你也拖下去。

---

## 冲突识别

**我的目标：** 拿回八千块
**陈姐的目标：** 一分不退

**表面在争：** 押金退不退
**实际上在争：** 我这样一个外地来的小姑娘，你能不能欺负得了

---

## 场景清单

**冲突一：要钱被拒（5分）**

→ 小林搬走那天去退押金。陈姐说大房东要收房，她也没钱
→ 小林掏合同。陈姐不看："你去告我啊"
→ 小林说不过她。钱没拿到。拉着箱子站在小区门口

**冲突二：居委会和稀泥（3分）**

→ 小林去居委会。大姐听了两边的话，说"各退一步，她退你四千"
→ 小林说凭什么
→ 大姐说"你一个外地小姑娘，别把事情搞太大了"
→ 小林回到出租屋，坐在床上，看着那堵隔断墙

**冲突三：举报（4分）**

→ 隔壁小刘告诉她：陈姐那几套房子有两套是违规隔断的
→ 小林去了房管局。填了举报表。匿名
→ 三天后陈姐打电话来，声音不对了："你非要这样？"
→ 小林说："八千退了，这事就完。"
→ 陈姐说好。第二天没转。第三天也没转。她在拖

**冲突四：陈姐老公上门（7分）**

→ 小林下班回来，看到门口站着一个人。陈姐老公
→ 他靠在楼梯扶手上，嘴里叼着烟。看到她来了，没让路
→ "你举报的？"
→ 小林没说话
→ 他把烟头摁在墙上，烟灰掉在她脚边
→ "小姑娘，你知不知道你在搞谁？"
→ "她欠我钱。"
→ 他笑了。不是笑她的话——是笑她还在说"欠钱"这件事。他觉得她还没搞清楚自己在跟谁搞
→ 他走了。走之前回头说了一句："你那个举报，最多罚点钱。下次就不是罚钱的事了。"
→ 小林站在楼道里，灯坏了。她站了很久才拿钥匙开门

**冲突五：老刘报信（4分）**

→ 楼下修车铺的老刘叫住她："你是不是跟陈姐那边搞上了？"
→ "她欠我钱。"
→ 老刘压低声音："他老公以前在里面待过五年。打残过人的。"
→ 小林没说话
→ 老刘说："你自己小心。"
→ 小林上楼的时候，脚步比平时轻。不是怕吵到人——是怕碰到有人站在门口等她

**冲突六：被迫反击（8分）**

→ 第二天，小林没去上班。她去了陈姐的小区——不是陈姐住的那栋，是另一栋
→ 她找到陈姐那辆电动车。从包里掏出一把螺丝刀。蹲下来
→ 她把前后两个轮胎的气放了。然后站起来，把螺丝刀放回包里
→ 她走回自己的出租屋，关上门。手指还在抖
→ 不是怕。是她这辈子没干过这种事
→ 她坐在床边，看着那双放气的脏手。然后站起来洗手。洗了三遍

**冲突七：陈姐怕了（6分）**

→ 当晚陈姐打电话来。语气不一样了，没有那么硬了
→ "你是不是动我车了？"
→ 小林没承认也没否认："你要不要试试看，下次是轮胎还是别的？"
→ 陈姐沉默了很久
→ "八千我没有。五千。明天给你。"
→ "八千。一分不能少。你拖一天，我多去一次。"
→ 陈姐挂了电话。第二天中午，八千块打到小林账上了

**冲突八：尾声（3分）**

→ 小林坐在ATM机前，看着余额里多出来的八千块
→ 她给陈姐发了一条消息："收到了。房管局那边我什么都没说。"
→ 陈姐没回
→ 她把手机放在桌上。去洗手
→ 洗到第二遍的时候，她看着镜子里的自己
→ 她想起老刘说那句话——"他以前在里面待过五年，打残过人的"
→ 八千块回来了。但她用的方式，是用螺丝刀放了一个人的轮胎。是威胁。是"你狠我更狠"
→ 她擦干手，走出洗手间
→ 窗外在下雨。上海进入梅雨季了，要下一个月的雨

---

## 终果

她从讲道理的人，变成了放轮胎气的人。钱拿到了。但手不干净了。

**弧光：** 讲道理→举报→放轮胎气。不是变强了，是变狠了。

**主题印证：** 「你逼一个人走到绝路，就别怪她把你也拖下去。」——印证。

**余韵：** 钱到手了。她洗了三遍手。
