# 浮刻日常 使用说明

## 这是什么

浮刻日常是一个提醒工具。你把想记住的东西放进去，它会在你设定的时间弹出来提醒你。

不是为了管理时间，不是为了提高效率。只是帮你记住——那些你觉得重要、但容易忘掉的事。

## 人生主线

人生主线是你持续关注的方向。

它可以是一句话、一个提醒、一个问题。比如"我到底在想要什么"、"放下想要控制"。

设定后，每天会自动弹出两次提醒。不是催你做事，只是让你看到它。

每个主线下面可以添加路标和现状：

- **路标**：你想要达到的状态
- **现状**：你现在的情况

路标用金色标记，现状用蓝色。不是用来对比差距的，只是把两个都摆出来，让你看见。

## 定时提醒

设定时间，到点弹出提醒。

- 支持选择多个时间点
- 可以设定为全天提醒（不指定具体时间）
- 提醒方式：通知栏通知 + 悬浮窗弹出

## 悬浮窗

提醒触发时，屏幕上会弹出一个悬浮窗，显示内容。

- 不需要打开App就能看到
- 看完后可以滑动关闭
- 可以设置提醒后自动关闭悬浮窗

## 随机弹出

除了定时提醒，还可以设置随机弹出。

每次解锁手机或打开App时，有一定概率弹出你设定的内容。目的是在你没有预期的时候，让你重新看到它。

概率可以在设置里调整。

## 答题卡

答题卡是一张卡片，上面写着一个问句，每次弹出时让你回答。

默认提供了三个问题：

1. 我能不能放下这个感觉？
2. 我愿不愿意放下？
3. 我什么时候放下？

你也可以自定义问题。弹出时看到问题，自己心里回答就行，不需要在App里填写。

## 图片

每条内容可以关联图片。在子列表右侧点击图片区域，可以添加图片。

- 支持裁剪（自由比例、1:1、4:3、16:9、9:16）
- 悬浮窗弹出时可以轮播显示图片
- 轮播时间和动画效果在设置里配置

## 设置

- **悬浮图标**：是否显示悬浮窗提醒
- **提醒后自动关闭**：弹出一段时间后自动消失
- **随机弹出概率**：解锁时触发的概率
- **轮播间隔**：图片轮播的速度
- **轮播动画**：淡入淡出/滑动/缩放/无
- **数据备份**：导出数据库文件
- **检查更新**：手动检查新版本

## 数据

所有数据存在手机本地，不上传任何服务器。备份功能可以导出数据库文件。
