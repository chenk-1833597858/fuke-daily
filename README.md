# 浮刻日常 — 使用指南

> **你所寻求的，从未离开你。** —— 莱斯特·利文森

---

## 这个App是什么

浮刻日常是一个**提醒工具**，但它不是普通的待办清单。

释放法的核心发现是：**一切限制都来自我们内在的想要——想要控制、想要认可、想要安全。** 当这些"想要"被释放，剩下的就是本来的自由。

但人容易忘。释放了一次，下一秒又被旧模式拉回去。不是释放不管用，是**忘了自己在释放**。

浮刻日常做的事很简单：**在你忘了的时候，把你想看到的，放到你眼前。**

---

## 核心概念

### 人生主线

这是最重要的功能。

一条主线 = 一个你持续释放的方向。比如：

- **我是否在试图控制这个局面？** → 释放想要控制
- **我在寻求谁的认可？** → 释放想要认可
- **我在恐惧什么？** → 释放想要安全

创建一条「人生主线」类型的列表，把你的释放方向写进去。App会在**每天两次**自动弹出内容提醒你——不是催你做事，是帮你**想起你已经在释放的路上**。

### 路标（子列表）

主线下可以添加子列表，作为更具体的路标。比如：

- 主线：「释放想要控制」
  - 路标：「工作中想控制同事的想法」
  - 路标：「想控制伴侣的反应」
  - 路标：「想控制结果不如预期时的焦虑」

### 现状

子列表里的具体条目，就是你当下的真实状态——不用美化，如实记录：

- 「同事没回消息，我感到烦躁」
- 「伴侣说了那句话，我瞬间想反驳」

**记录本身就是在看。看到了，才能释放。**

## 功能说明

### 📋 列表类型

| 类型 | 用途 | 适合释放法的场景 |
|------|------|-----------------|
| **选择** | 手动点选显示内容 | 放一组释放问句，随时刻意选取 |
| **随机** | 随机展示一条 | 打破惯性，让该看到的浮出来 |
| **答题** | 问答卡片 | 自问自答：「我现在能放下这个感受吗？」 |
| **人生主线** | 每日两次自动触发 | 核心释放方向的持续提醒 |

### 🔔 定时提醒

设定时间段，App会在设定时段内**随机弹出**内容。

不是固定时间——固定时间心智会预判，随机弹出才能**穿透惯性思维**，在你真正需要的时候出现。

支持全天模式和自定义时段。

### 🫧 悬浮窗

一个始终在屏幕上的小图标，点击后展开显示当前内容。

**你不需要打开App，它就在那里。** 这就是浮刻日常的含义——浮在一切之上，刻在当下之中。

悬浮窗支持图片轮播，可以把对你有意义的画面放在最显眼的位置。

### 🖼️ 图片

每个子列表可以关联图片。点击子列表右侧的图片占位 → 进入图片管理 → 添加图片 → 选择裁剪比例。

图片会显示在悬浮窗中，轮播展示。轮播速度可在设置中调整。

### ⚙️ 设置

- **主题**：温暖 / 暗夜
- **权限获取**：悬浮窗、通知、闹钟、自启动、电池优化
- **轮播速度**：悬浮窗图片轮播间隔
- **检查更新**：App启动自动检查，也可手动检查

## 推荐用法

### 给初学者的设置

1. 创建一条**人生主线**，标题写你最大的释放方向
2. 加2-3个子列表作为路标
3. 每个子列表写3-5条现状
4. 开启悬浮窗

就这样。剩下的交给App。

### 每日流程

- **早晨弹出**：看到主线提醒 → 感受当下的"想要" → 能放下吗？
- **白天悬浮窗**：随机看到路标 → 检查：我在抓取什么？→ 释放
- **晚间弹出**：回顾这一天 → 还有什么没释放的？→ 放下

### 进阶：释放问句卡

创建一个**答题类型**的列表，写上释放法的核心问句：

1. 我现在能放下这个感受吗？
2. 我愿意放下它吗？
3. 我什么时候放下它？

答题时，不是在"做题"，是在**给自己一个释放的契机**。每个问题的答案不重要，重要的是你在问的瞬间，已经有了放下的可能。

## 最后

莱斯特说：「**你不需要获得任何东西，你只需要放下。**」

这个App不会帮你释放，但它会帮你**记住**——你已经拥有你所寻求的一切，只是还有些东西没有放下。

当悬浮窗弹出，当提醒响起，不需要做任何事。只需要**看一眼**。

看一眼就够了。
