# 冲突参考手册

## 一、冲突的定义

**冲突不是"分歧"，不是"方法不同"，不是"意见相左"。**

冲突是一个人试图让另一个人**不舒服、不爽、不好过**——有意的、主动的、持续的。

### 冲突的三种形态

| 形态 | 特征 | 例子 |
|------|------|------|
| **显性冲突** | 面对面，说话不客气 | "你是不是有病" / "你写的什么垃圾" |
| **隐性冲突** | 表面客气，暗地较劲 | 当着面说"辛苦了"，背后跟别人说他写的方案一文不值 |
| **冷冲突** | 不说话，不合作，不接茬 | 故意不回传讯符 / 他说话时她低头擦杯子假装没听到 |

三个形态可以互相转化。显性冲突谈崩了会变成冷冲突（表面不说话），冷冲突积累够了会爆发成显性冲突。

### 冲突的制造手法（不只是对话）

| 手法 | 定义 | 例子 |
|------|------|------|
| **嘲讽** | 用表面肯定的方式表达否定 | "守则第十七条背得真熟啊"——表面夸你记性好，实际说你不会动脑子 |
| **下黑手** | 表面不冲突，暗中制造损失 | 他值班时报给执事堂"未见异常"，实际少报了一截灵脉的隐患——让下一个人背锅 |
| **污蔑** | 把没说过的说成你说的，把没做过的说成你做的 | "你上次不是说灵脉不用查那么细吗？"——他根本没说过 |
| **假话当真** | 反复说一个不存在的事实，让第三方相信 | 在长老面前"恰好提了一句"言肃上次巡查迟到了——其实没迟到，但说多了别人就信了 |
| **忽视** | 故意不回应、不接话、不合作 | 言肃递交接单，顾自横不接，放在桌上走开——这比说"我不签"更侮辱人 |
| **比较** | 用第三方压你 | "上次王长老带队，哪像你这么搞"——王长老根本不在这件事上 |
| **截话** | 打断对方的关键发言 | 言肃刚开口说"我认为应该先……"顾自横就对长老说"我觉得直接下去就行"——根本不让他说完 |
| **过度配合** | 用过度服从制造讽刺 | 言肃说"子时要交接"，顾自横提前一个时辰就在门口等着——表面配合，实际是说"你定的规矩真可笑" |

**核心原则**：冲突不一定靠吵架实现。吵架是冲突的结果，不是冲突本身。沉默、配合、微笑——都可以是冲突的武器。

## 二、"同事看不顺眼"的冲突分类

### 1. 观念之争（最体面，但最无力）

> "我觉得应该先上报" vs "我觉得应该先查了再说"
> → 力量等级：★☆☆☆☆（容易变成学术讨论）

**不推荐单独使用**。必须配上更低的招数，比如：
- 观念之争 + 他在你说话时翻了个白眼
- 观念之争 + 他在你说完后对旁人笑了一下

### 2. 优先级之争（中等级别）

> "你的那点破事凭什么排在我前面"
> → 力量等级：★★★☆☆

例子：言肃觉得巡查排班要以灵脉走向定，顾自横坚持先查他用得顺手的那段——不是方法不同，是他**不承认你的方法有道理**。

### 3. 面子之争（强烈级）

> "你为什么要在长老面前让我不好看"
> → 力量等级：★★★★☆

不是方法之争，是**尊严之争**。
- 他在三个人面前说你上次写错了
- 他明知道你刚被批评过，还在同一个会议上提了同样的问题
- 他故意在你刚说完话之后说"那我说说我的看法"

### 4. 边界之争（最强烈级）

> "你凭什么动我的东西 / 管我的事 / 越我的界"
> → 力量等级：★★★★★

这是同事冲突的终极形态——不涉及正事，纯属"我不爽你踏入我的领地"。
- 他碰了你桌上的符纸（没弄坏，就是碰了）
- 他替你回答了长老的提问（你应该回答的那个问题）
- 他在你的巡查记录最后一页翻了一下（没改，就是看了）

**注意**：边界之争的核心不是"损失"——他什么都没破坏，什么都还在。但你感受到了冒犯。

## 三、冲突的节奏：从轻到重，不可回头

不是一上来就撕破脸。冲突有一条**升级路径**：

```
看不顺眼（内心活动）→ 小动作（他关门重了）→ 隐性攻击（不接话） 
→ 显性攻击（嘲讽）→ 伤害性行为（下黑手/污蔑）→ 不可挽回（真正的损失）
```

**每个阶段都要对应一个具体的场景。** 不能跳过。

```
第1天：言肃关门的声音比平时大了一点。顾自横没说什么，但抬头看了他一眼。
→ 层级一：关门声（显性？还是我想多了？）——冲突还没坐实，但读者感觉到了

第7天：言肃对顾自横说"灵脉入口好走，你按这个方向走"，顾自横嗯了一声，反方向走了。
→ 层级一：他故意不按他说的走——冲突坐实了，不需要大吵

第15天：言肃发现顾自横提报的巡查记录里，故意把他说过的话删了一句。
→ 层级一：篡改——不是"方法不同"，是攻击
```

**关键规则**：同一件事不能重复两次。如果场景1的冲突是"关门重了"，场景2就不能还是"关门重了"——必须升级。

## 四、"冲突" 的七个实际案例（场景级）

以下每个案例都是一个场景，有层级一对抗，有具体行为：

### 案例1：截话

长老问："此次灵脉巡查有何异常？"
顾自横张嘴要答。
言肃："有。第三层灵气波动，我已记录在案。"——直接替他说了。
顾自横闭上嘴，看着他写完。
言肃写完抬头："你补充？"
顾自横："你都说了，我没什么好补充的。"
→ 冲突形态：显性（我替你说，你就不用说了）

### 案例2：背后一句

言肃不在场时，有弟子来问顾自横："上次那条灵脉的巡查报告怎么写的？"
顾自横："言肃师兄写的。他做事仔细，你按他写的交就行。"——表面夸，实际说"他写的，我不负责"。
等言肃知道这句"推荐"时，那份报告已经因为太保守被长老打回来了。
→ 冲突形态：隐性（表面配合，暗地撇清）

### 案例3：过度配合

言肃排了轮值表，子时交接。
顾自横戌时就到了，坐在值守屋里等了一个时辰。言肃到时，顾自横说："师兄定的时候，我早到一个时辰，够配合了吧？"
言肃没说话——他知道这不是配合，是嘲讽。
→ 冲突形态：显性（过度服从本身就是讽刺）

### 案例4：假传话

两人共同负责的事务，长老问进展。
言肃问顾自横："你那部分做完了吗？"
顾自横："快了。"
言肃回头跟长老说："他那边快了，不影响整体进度。"
结果第二天顾自横说："我还没开始。言肃师兄说快了？我说的是快了——但不一定是今天。"
长老看向言肃。言肃无话可说。
→ 冲突形态：污蔑（他利用了"快了"的歧义，让你在长老面前失信）

### 案例5：选择性忘记

言肃交代："明日卯时，第三层洞口汇合。"
第二天卯时，言肃在洞口等了半个时辰。顾自横没有来。
事后顾自横说："我以为你说的是东面那个口。你说了吗？可能我没听清。"
→ 冲突形态：隐性（说没听到——你没法证明他听到了。这就是故意的最高境界）

### 案例6：在第三方面前让你难堪

聚会上，有弟子请教言肃一份符文的写法。
言肅正要答。
顾自横在旁边说了一句："言肃师兄的符文功底是宗门出了名的。不过这次灵脉的事，他好像没看出什么来。"
弟子听完，不知道该不该继续问了。
言肃放下杯子。
→ 冲突形态：显性（当众点你的失败，不让你在弟子面前有尊严）

### 案例7：延迟合作

言肅需要顾自横签字才能推进下一步。
他把单子递过去。顾自横看了一眼，说"知道了"，没有签。
第二天。言肃问："签了吗？"
顾自横："哦，我想着等一下再签。"
第三天。言肃再问。顾自横："马上。"
第四天。言肃不问了。自己想办法绕过了这个签字环节——但多花了三天时间。
→ 冲突形态：冷冲突（不说不签，就是不签——让对方承受后果）

## 五、冲突与"打架"的区别

**不能打架** ≠ **不能造成伤害**。

| 可以做的事 | 不能做的事 |
|-----------|-----------|
| 推搡一下肩膀（没动手，就是路过时肩膀撞过去了） | 一拳打过去 |
| 把他的符纸从桌上扫到地上，说"哎呀不是故意的" | 撕了他的符纸 |
| 在人前让他下不来台 | 在背后造谣说他是叛徒 |
| 在他的巡查日志上发现一个小错，当着长老的面'好心'指出来 | 捏造他根本没犯的错 |

**原则**：物理伤害止于"擦肩而过"。但心理伤害没有上限。

## 六、冲突与"误会"的区别

**冲突不是误会。**

| 冲突 | 误会 |
|------|------|
| 他故意把你的交接单放错地方 | 他不小心放错了 |
| 他明知你着急用还拖着不签字 | 他不知道你着急 |
| 他当众翻你的旧账让你难堪 | 他无意中提到了那件事 |

**检验标准**：如果把话说开，冲突会消失吗？
- 会消失 → 是误会，不是冲突。删掉或改成真正的冲突。
- 不会消失 → 是冲突。他已经决定了要让你不好过，说开了也没用。

**为什么这个检验重要：** 误会可以靠"一次对话"解决——那就不是好冲突。好的冲突是**即使你知道他是故意的、你也知道他知道你知道——但你还是没办法。** 这才是人际冲突的持久张力。

## 七、"修仙同事"版冲突的特殊性

修仙日常冲突和普通职场冲突的不同在于：

| 维度 | 普通职场 | 修仙世界 |
|------|---------|---------|
| 物理距离 | 下班各回各家 | 灵脉同住三个月，躲都躲不开 |
| 权力干预 | HR可以调解 | 长老不管这种小事——"你们自己解决" |
| 后果 | 最多绩效差 | 分灵脉资源时你少一份，修为就慢一步 |
| 散伙方式 | 换个组 | 可能一辈子都要在同一个宗门 |
| 冲突工具 | 邮件抄送、拉群 | 传讯符、巡查日志、排班表、丹药房排队、食堂抢座 |

**"躲不开"** 是修仙同事冲突最特殊的地方。普通人可以避开讨厌的同事——修仙不行。一闭关出来发现还是同一个人，一根灵脉上两个人轮流值守三年。这种"不得不面对"的压抑感，是冲突发酵的温床。

## 八、高强度冲突示例（真实案例改编）

以下示例来自真实事件，展示了"争的是人命/尊严/一生"级别的冲突。注意每一条线都是**物理的、当下的、不可逆的**——没有人"冷静一下"，没有人"改天再说"。

### 示例1：家暴·女儿报警抓父亲

冲突本质：暴力循环——打完认错，认完再打。逃不掉。

```
父亲酗酒回来砸东西，母亲说了一句"离婚吧"
→ 父亲抓起玻璃杯砸地，一把揪住母亲头发推倒，耳光扇到嘴角出血
→ 五年级的女儿冲出来拉父亲，被一耳光扇到耳聋几秒
→ 女儿抓起手机冲出家门报警
→ 警察来了训诫，父亲保证不再犯
→ 警察走后，风头一过，暴力照旧
```

**强度：** 8分。最狠的不是那一耳光——是"照旧"两个字。

### 示例2：六合彩·中奖暴毙

冲突本质：追了一辈子的东西到手了，人死了。

```
一个在县城工厂打工的男人，痴迷地下六合彩
→ 老婆劝，吵架；孩子学费不够，吵；父母生病，还是吵
→ 他欠了一屁股赌债，老婆要离婚
→ 某天他中了43万——他一辈子没见过这么多钱
→ 晚上喝酒庆祝，激动过度，心肌梗塞
→ 当场死了
→ 43万在桌上，人在地上
→ 老婆来收尸，看了一眼钱，说了一句："你是被这钱害死的"
```

**强度：** 8分。冲突不在人和人之间——在一个人和命运之间。追上了，也死了。

### 示例3：乡村灭门·一句口角烧了四条命

冲突本质：官法不管，自己来。以命偿命。

```
天贵母亲跟王家在牌场吵了一架，两家因菜地分界线积怨多年
→ 王家老伴："等着见报应"
→ 天贵家当晚起火，父母妻儿四口全烧死
→ 警方说"电线老化"排除了他杀，王家走个过场就放了
→ 天贵回家看到四口棺材，上吊被邻居救下
→ 天贵去王家哭闹要说法，王家只送了一个花圈
→ 王家大门紧闭，老伴故意大声说："不做亏心事，不怕鬼敲门"
→ 几天后王家两人身中数刀死于屋内，旁边放着蜂窝煤炉子
→ 5岁孙子被盖着父亲和爷爷的衣服睡在客厅，捡回一条命
→ 凶手被抓，供认不讳："拿刀那一刻就什么都不怕了"
```

**强度：** 10分。穷人的公道不在法律那里。

## 高强度冲突的共性特征

以上三个案例的共同点：

1. **有人流血/砸东西/死**——不是"他沉默了一下"，是生理层面的反应
2. **每一轮都在往对面软肋上捅**——没有废话，没有解释，没有铺垫
3. **"我不能走"的理由**——走了家就散了/证据消失了/人白死了
4. **打完以后，有个人变了**——不是"和好了"，是"回不去了"
5. **不可逆**——耳光扇出去了、钱到手了人也死了、刀捅进去了。不能撤回

**写冲突前先问自己：这一场有没有人会真的疼？没有人疼的冲突，删掉。**

### 示例4（都市版）：铺面烧了

冲突本质：她报了警，没证据。她哥用她的方式替她讨了回来。

```
两家人在同一条街上开了十五年店。一家卖五金，一家卖早餐。
→ 早餐店老板娘说五金店的货堆到了她门口
→ 五金店老板娘说那地方是公家的
→ 吵过三次。城管来过两次。谁都没服
→ 三天后，五金店半夜起火。卷帘门烧化了，二十万的货没了
→ 消防说电气线路短路。五金店老板娘不信——她早上拉了所有电闸
→ 她去派出所报案。那一段监控刚好坏了
→ 她说："是她烧的。她早上跟我说过——你等着。"
→ 民警说："三个字不能当证据。"
→ 她哥来了。看到她蹲在烧焦的卷帘门前
→ 她哥买了一条烟，找到了那个电工
→ 电工说前天晚上有人让他去把那段监控线剪了，给了两千块钱
→ 她哥蹲在那家早餐店门口，抽了一根烟
→ 早餐店老板娘端着水出来泼在地上——溅到他鞋上："让让。我要做生意了。"
→ 当天晚上，早餐店的卷帘门被人用钢筋别开了
→ 煤气灶的两个旋钮拧到了最大。窗户关着
→ 第二天早上，她男人开门点火——火从厨房蹿了出来
→ 全身百分之六十烧伤。进了ICU
→ 民警来了："你干的？"
→ 她哥坐在自己烧焦的铺面前："你有证据吗？"
```

**强度：** 9分。最狠的不是那把火——是她哥说出"你有证据吗"的时候。她把警察教她的那句话还了回去。
