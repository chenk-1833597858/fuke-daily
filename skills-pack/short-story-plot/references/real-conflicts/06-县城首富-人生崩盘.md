# 冲突案例06：县城首富·40岁人生崩盘

**来源：** 网易「人间theLivings」——《县城首富，40岁人生崩盘》

## 冲突概要

从摆摊做到几个亿的县城首富，盲目扩张导致资金链断裂，妻离子散，债主上门，蹲了三年监狱。出来后回到县城菜市场卖菜，曾经的朋友假装不认识他。

## 交锋链

```
从摆摊卖衣服起家，十年做到几个亿
→ 野心膨胀，同时开三个楼盘
→ 资金跟不上，借高利贷
→ 利息滚利息，雪球越滚越大
→ 银行抽贷，资金链断裂
→ 供应商堵门讨债
→ 工人拉横幅讨薪
→ 妻子带着孩子走了
→ 借过钱的"朋友"一个个消失
→ 法院传票、资产冻结、破产清算
→ 40岁那天，他站在自己盖的烂尾楼顶上想跳
→ 没跳。蹲了三年监狱
→ 出来那天，没人接
→ 回县城，在菜市场租了个摊位卖菜
→ 以前的老朋友来买菜，看了一眼，假装没看见走过去了
→ 他低头继续称菜，手没抖
```

## 三层冲突挖掘

- **表面在争：** 钱能不能还上
- **实际上在争：** 从"人生赢家"到"卖菜的老头"，你还能不能活下去——最大的敌人不是债主，是镜子里的自己
- **输不起什么：** 他输不起"面子"——但面子先输光了；朋友输不起"被他借钱"

## 冲突强度

**8/10** — 没有肢体冲突。最大的一刀是"假装没看见"——从县城首富到连招呼都不值的人，这落差比破产本身更致命。

## 写作启示

"人生崩盘"型冲突：不是两个人对抗，是命运和一个人的对抗。周围所有人都在撤退，才是真正的绝境。
