# 冲突案例11：被裁·中年男人的送外卖之路

**来源：** 网易「人间theLivings」——《西安代驾圈，挤满了失业的经理人》

## 冲突概要

40岁的白领经理被公司"降本增效"优化掉，屡次求职失败后开始跑外卖。对着家人强装笑脸、对着前同事隐瞒现状、在系统规则和罚款之间挣扎。冲突不是跟某个人，是跟整个"体面"的崩塌。

## 交锋链

```
领导把他叫进会议室，说"你负责的工作不是刚需"
→ 他在工位上磨蹭到七点多才走
→ 路上突然下大雨，淋成落汤鸡
→ 回家孩子扑上来叫爸爸，他挤出笑容说"让你见识见识落汤鸡"
→ 老婆问今天怎么了，他沉默了很久才说出被裁了
→ 老婆说"干不成就不干了，再也不用受谁的脸色了"
→ 他往嘴里塞了瓣苹果，酸甜难辨
→ 开始投简历，10份没回音，30份石沉大海
→ 一周投60份，才换来3个面试，全挂了
→ 开始跑外卖，日收入最多200块
→ 母亲打视频问"最近忙啥呢步数那么多"，他说"公司拓展训练"
→ 挂完电话，关了微信运动，出门跑单
→ 有一单因"虚假送达"被扣50块——心里直骂娘
→ 决定：明天继续跑
```

## 三层冲突挖掘

- **表面在争：** 能不能找到工作、能不能还房贷
- **实际上在争：** 一个男人跟自己"体面"的那道坎——从坐在办公室的经理，到在街上被系统罚款200块一天的外卖员，最难的不是身体累，是对家人说谎的时候
- **输不起什么：** 输不起"让家人看出来我撑不住了"；输不起"这辈子就到这了"

## 冲突强度

**7/10** — 没有跟任何人的正面对抗。但最大的敌人是"自己能不能接受自己变成这样"。那句"公司拓展训练"是整个故事最狠的一句——对着亲生母亲说谎，不是怕她知道，是怕她知道了他绷不住。

## 写作启示

"跟命运对抗"型的冲突模板。没有具体的对手，但处处都是对抗——跟系统对抗（被扣50）、跟面子对抗（谎称拓展训练）、跟现实对抗（40岁重新求职）。适合写"底层人物挣扎"类剧情。
