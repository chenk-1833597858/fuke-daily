# 冲突案例05：重组家庭·50年偏心

**来源：** 网易「人间theLivings」——《重组家庭，为"偏心"争了50余年》

## 冲突概要

继母带来的孩子和父亲亲生的孩子，从五六岁争到五十多岁。争的不是钱，是"你到底有没有把我当亲生的"。

## 交锋链

```
继母进门那年，两个孩子都五六岁
→ 分西瓜，继母切给亲生那块更大
→ 上学，给亲生买了新书包，继子背旧的
→ 过年，给亲生压岁钱多了一倍
→ 工作，托人给亲生孩子安排了国企
→ 继子什么都靠自己
→ 几十年里无数次"小事"，但每一件都刻在骨头里
→ 分家产那晚，继母把房子留给了自己孩子
→ 继子终于爆发："你从来没把我当过家人"
→ 继母："我对你还不够好？"
→ 继子："好？你亲生的孩子上了大学，我呢？"
→ 继母沉默
→ 从此两家人形同陌路
→ 五十多岁了，还在为"当年那个西瓜"耿耿于怀
```

## 三层冲突挖掘

- **表面在争：** 分财产不公平
- **实际上在争：** "你到底有没有把我当过亲生的"——一个动作重复五十年，就是答案
- **输不起什么：** 继母输不起"被人说不公平"；继子输不起"被当作外人过一辈子"

## 冲突强度

**7/10** — 不是一次大爆发，是五十年积累的小伤口。每一个"忘了你"都是一刀。

## 写作启示

最长效的冲突不是雷鸣电闪，是一点一点的"不公正重复"。
