# 冲突案例09：六合彩·中奖暴毙

**来源：** 网易「人间theLivings」——《地下六合彩赌徒，猝死于中奖43万那晚》

## 冲突概要

一个县城工厂工人痴迷地下六合彩，老婆要离婚，孩子学费不够，欠了一屁股赌债。某天中了43万——他一辈子没见过这么多钱。当晚喝酒庆祝，心肌梗塞，当场死了。

## 交锋链

```
一个普通工人，迷上了地下六合彩
→ 老婆劝他别赌了，他骂回去
→ 孩子学费不够，老婆问他要钱，他说没钱
→ 父母生病住院，他连医药费都拿不出
→ 老婆要离婚："你跟六合彩过吧"
→ 他跪下来求老婆再给他一次机会
→ 转身又去买了一注
→ 中了！43万——他这辈子都没见过这么多钱
→ 晚上叫了几个兄弟喝酒庆祝
→ 喝到兴头上，他突然捂住胸口，从椅子上滑下去
→ 心肌梗塞
→ 当场死了
→ 43万现金在桌上，人在地上
→ 老婆来收尸，看了一眼那些钱
→ 说了一句："你是被这钱害死的"
```

## 三层冲突挖掘

- **表面在争：** 赌不赌、还不还债
- **实际上在争：** 一个穷了一辈子的人，承受不了命运突然给的好——他的命比那43万薄
- **输不起什么：** 他输不起"翻身的机会"；老婆输不起"这个家"

## 冲突强度

**8/10** — 最大的戏剧张力在"中了"和"死了"之间那几小时——他以为命运终于对他笑了，其实是命运在跟他开最后一个玩笑。

## 写作启示

"愿望实现了反而毁灭你"是最有层次感的冲突——不是求而不得，是求而得到后承受不住。
