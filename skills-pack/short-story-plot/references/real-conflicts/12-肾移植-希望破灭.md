# 冲突案例12：肾移植失败·30万打水漂

**来源：** 网易「人间theLivings」——《30万换的肾，只在我体内活了30天》

## 冲突概要

一个透析病人等了很久终于等到肾源，花光积蓄做了移植手术。术后感染、移植肾坏死切除、瘫痪、面瘫、左耳失聪、体重从100斤掉到60斤——最后回到透析。从希望的天堂摔回绝望的原地。

## 交锋链

```
透析多年，终于等到匹配的肾源
→ 全家凑了30万做移植手术
→ 术后还没能下地几天，就因感染又回院
→ 移植肾坏死，切除
→ 成为ICU常客
→ 瘫痪了——原本直立站着的人躺下了
→ 面瘫，做不出表情，无法哭笑
→ 无法正常说话，吃东西张不开嘴
→ 左耳聋了，左眼视力模糊
→ 入院时100斤，出院时60斤
→ 回到透析——跟手术前一样，但人已经不是那个人了
→ 但还活着。作者说："对我来说这已经是奇迹了"
```

## 三层冲突挖掘

- **表面在争：** 能不能治好病
- **实际上在争：** 命运给了你一个希望，再把这个希望砸碎给你看——给了肾又拿走，给了希望又让回到原点，比从来没给更残忍
- **输不起什么：** 输不起"三十万打了水漂"；输不起"熬了那么久最后还是回到透析"

## 冲突强度

**7/10** — 不是人和人之间的对抗，是人和命运。最狠的不是手术失败，是"出院时60斤，但还活着"——你把命保住了，但你失去了一切能证明你活着的功能。

## 写作启示

"希望-绝望"反转模板。不是一次反转，是一连串的：等到了→手术了→感染了→切除了→瘫痪了→失聪了→回到原点。每一层都让你以为是最低点了，但下一层更狠。适合写"拼命追求一个东西，追到了才发现代价比得到更大"的剧情。
