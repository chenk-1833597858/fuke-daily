# 冲突案例10：农村父子·香火执念

**来源：** 网易「人间theLivings」——《困在香火执念中的农村父子，走上了脱贫路》

## 冲突概要

父亲一辈子就一个念头：儿子必须生儿子。儿媳妇生了女儿，父亲脸垮了。催二胎，还是女儿。父亲逼儿子离婚，儿子不肯。父亲绝食："你断香火我就不吃饭。"

## 交锋链

```
父亲一辈子就想抱孙子——儿子必须生儿子
→ 儿媳妇第一胎生了女儿，父亲什么话都没说，脸拉了一天
→ 儿子哄父亲："再生一个就好了"
→ 二胎怀上了，父亲天天烧香
→ 又生了女儿
→ 父亲把孩子抱到一边，坐了大半天没动
→ 之后父亲开始催："再生一个"
→ 儿媳不肯了："我生两个了还不行？"
→ 父亲不说话，开始绝食
→ 儿子跪在父亲面前："爸，现在谁还管这个"
→ 父亲："你断了我香火，我死了怎么面对列祖列宗"
→ 儿子："那我两个女儿算什么？"
→ 儿媳妇抱着两个孩子在门口站着，没有进门
→ 儿子跪在父亲面前，身后是妻子和女儿——他夹在中间，一句话说不出
→ 家里安静得像没人住
```

## 三层冲突挖掘

- **表面在争：** 生不生儿子、要不要三胎
- **实际上在争：** 父亲怕自己死了以后没人记得他——香火断了就是"白活了一辈子"；儿子是想做"好儿子"还是"好丈夫好爸爸"的挣扎
- **输不起什么：** 父亲输不起"绝后"；儿子输不起"对不起妻子和女儿"；儿媳输不起"在这个家永远是个外人"

## 冲突强度

**8/10** — 没有打骂没有摔东西，但绝食这一招比什么武器都狠——父亲用"我把自己饿死"来逼儿子，儿子跪着哭，老婆孩子站门口看着。

## 写作启示

传统vs现代的价值冲突模板。父亲不是坏人，儿子不是不孝——但两个对的世界撞在一起，总要碎一个。
