# 冲突案例01：家暴·女儿报警抓父亲

**来源：** 网易「人间theLivings」——《我的母亲，只想要一间没有家暴和追债的屋子》

## 冲突概要

父亲酗酒、家暴、欠债；母亲从年轻漂亮被拖到花白头发；女儿从小目睹暴力，在恐惧中长大。

## 交锋链

```
父亲酗酒回来砸东西，母亲说了一句"离婚吧"
→ 父亲抓起玻璃杯砸地，一把揪住母亲头发推倒，耳光扇到嘴角出血
→ 五年级的女儿冲出来拉父亲，被一耳光扇到耳聋几秒
→ 女儿抓起手机冲出家门报警
→ 警察来了训诫，父亲保证不再犯
→ 警察走后，风头一过，暴力照旧
→ 这样的循环重复了十几年
```

## 三层冲突挖掘

- **表面在争：** 离不离婚、打不打人
- **实际上在争：** 一个女人的一辈子被"婚姻"两个字死死绑住——她不是不想走，是走了活不下去
- **输不起什么：** 母亲输不起"女儿没有爸爸"这个标签；父亲输不起"老婆孩子敢不听我"的权威

## 冲突强度

**8/10** — 不是身体伤害的程度，是"逃不掉"的窒息感。每一次报警都像一拳打在棉花上，法律来了又走了，暴力又回来了。最狠的不是那一耳光，是女儿知道报警也没用。
