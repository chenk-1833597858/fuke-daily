# 冲突案例08：婚姻暗算·丈夫的遗书

**来源：** 网易「人间theLivings」——《我发现了丈夫的一封遗书：他要在死前夺走我所有财产》

## 冲突概要

妻子在整理丈夫书房时发现一封遗书——丈夫身患绝症，计划在死前把所有财产转移给初恋。她跟他十五年，陪他从零到有，他临终想的却是别人。

## 交锋链

```
妻子整理书房，在一本书里发现一封遗书
→ 遗书是丈夫写的：他要在死前把所有财产转给初恋
→ 公司股权怎么转、房子怎么卖、钱走哪张卡——写得清清楚楚
→ 妻子跟他十五年，陪他从零到有
→ 她拿着遗书在客厅坐到天亮
→ 第二天，她把遗书放到丈夫面前
→ 丈夫沉默了很久："你想怎么办"
→ 妻子说："离婚。你的钱我不要。但你欠我这十五年怎么算？"
→ 丈夫说："对不起"——结婚十五年，他第一次说这三个字
→ 妻子没哭。她站起来走了出去
→ 走到门口停了一下，想说点什么，没说
→ 门关上了
```

## 三层冲突挖掘

- **表面在争：** 钱归谁
- **实际上在争：** 她陪他十五年，他临终想的却是别人——那这十五年算什么？是投资？是搭伙？还是她自作多情？
- **输不起什么：** 妻子输不起"这十五年都是笑话"；丈夫输不起"死前要当个好人"的假象

## 冲突强度

**9/10** — 没有争吵、没有摔东西。最狠的是遗书上那些冷静的文字——"股权、房子、银行卡号"——每一个数字都在说：我从来没把她算进我的未来。

## 写作启示

"背叛+算计"是最高级别的冲突燃料：发现被最亲近的人背叛，而且对方连后路都安排好了。
