# 用户通过的案例骨架

## 案例1：「我不干了」（2026-06-18通过）

**核心特征：**

1. **第一人称「我」贯穿始终** — 不是「她」，是「我」
2. **开局价值观碰撞（8分）** — 他为啥说"那辞啊"那么轻飘飘？因为在他的世界里辞职不是大事。在我的世界里我卡里就两万块。这不是谁对谁错，是两个人活在不同的世界里。
3. **多战场** — 在家跟男朋友打(钱不重要vs我扛不住了)、在公司跟老板打(能者多劳vs你觉得我不值)、在电话跟妈妈打(忍忍就过去了vs你忍出什么了)
4. **没有非冲突场景** — 每一场都是真干，没有朋友问「你怎么了」这种注水
5. **曲线：高(8)→中(6)→中(6)→中(6)→高(7)→高(7)→低(3)** — 第一场最高(8分)，末尾反弹(7分)，尾声最低但最完整

**曲线模式：** 高-中-中-中-高-高-低

**硬门槛：** 前两场≥3 ✅ | 至少一场≥7 ✅ | 终果≥6 ✅ | 总分40分 ✅

**关键配角用法：**
- 江洲（男朋友）— 冲突对象（价值观碰撞：他觉得辞职很简单，她扛着经济压力）
- 陆总（老板）— 冲突对象（能者多劳的剥削者，最后一张牌是恐吓）
- 妈妈 — 对立面（两代人价值观冲突：忍 vs 不忍，回击是「你找了江洲那样的跟我找你爸有什么两样」）

**可复用的结构：** 家庭冲突2场 + 职场冲突3场 + 亲情冲突1场 + 尾声。家庭和职场交替打，不高强度重复。

## 案例2：「他用了三年，才学会看我」（2026-06-18通过，70分）

**核心特征：**

1. **和好结局** — 不是"她走了不回头"，是她看到他真的改了，选择回来。难点在于和好的说服力：他的改变必须可验证（推了项目、热水器细节、便签铅笔字），不是嘴上说说。
2. **开局价值观碰撞（6分→提至8分）** — 原版她直接走人，改后加了4轮攻防：她说「你每次都这么说」→他反击「我是在给这个家挣钱」→她点破「我是你老婆不是你项目的备注」
3. **多战场** — 家(书房/楼道)、电话(婆婆→妈妈→他)、出租屋门口、公司楼下、以前的家里、餐厅。同一个伤口（他不看她）被从6个角度戳。
4. **两大高潮场景在全篇后半段** — 场景7（冰箱便签三行铅笔字，9分）和场景8（「当三年前」+手指冰的比喻，9分）在后半段，但前期（场景1-3总分8+7+8=23分）已经撑住了读者的注意力，后半段的高潮是对前期建立的情绪的"还债"而不是"意外高潮"。
5. **冲突真实性验证** — 每个场景都有真对抗。朋友小鹿只在场景5一笔带过（推动剧情），妈妈在场景3有真实价值观碰撞（忍了一辈子了vs你忍出什么了），婆婆只在推动剧情。

**曲线：** 8→7→8→8→7→8→9→9→6

**硬门槛：** 前两场≥3 ✅ | 至少一场≥7 ✅（全场7分以上） | 终果≥6 ✅（场景8=9分） | 总分70分 ✅

**情绪曲线检查：** 前期需求不被满足（她走了，他还没看到）→中期松动（他开始改变）→后期被满足（他真的改了，她选择了回来）

**关键的"无声证据"技巧：**
- 场景7的冰箱便签 — 他不只用铅笔写了「已买」，还在旁边写了三行又划掉的内心独白：「她什么时候回来？」→「她会回来吗？」→「等她回来。」
- 这也是一种追踪——在空间上留下他心路历程的线索，让她(和读者)一步步发现他在她不在的时候是怎么过的。

**可复用的结构：** 前3场密集交锋建立情绪（书房→楼道→电话），中间2场推动改变（他认错→他真的改了），最后3场递进到和解（热水器→便签→当三年前）。不均衡分布，前重后也重但中间有呼吸空间。

## 案例3：「他见过一千个病人，唯独没看见我」（2026-06-20通过，40分）

**核心特征：**

1. **医疗背景** — 我是外科病房护士，他是外科医生，同院五年。利用同院场景制造「他经过晕倒的我却没有停」这种高密度冲突机会。
2. **开局价值观碰撞（8分）** — 深夜我把菜热了三次，他直奔书房。他说「我做了四台手术」，我说「五天没在家吃过饭了」。他问「那你想怎么样」——语气是「你又要闹」。价值观对立：他认为「救死扶伤是免死金牌」，我认为「看见我是婚姻的基本线」。
3. **同院设定优势** — 走廊晕倒他经过、护士长追喊「你老婆晕倒了」、他在天台一手病历一手说「我想离婚」——场景切换成本极低，每场都是真打。
4. **没有非冲突场景** — 7场只有6打1收，每一场都有人面对面互不相让。
5. **配角三线清晰** — 护士长（敲打者，2句台词完成功能）、我妈（代际对立，7秒沉默解决）、老病人家属（信息传递，唯一渠道）。
6. **意象系统简洁** — 排骨（他忘了的承诺）→ 报告（他翻了一堆没翻到）→ 栗子（他放了，我吃了）+ 报告（他对实习医生说了「不能放桌上等人翻」）。

**曲线：** 8→7→6→7→5→4→3

**硬门槛：** 前两场≥3 ✅（8+7） | 至少一场≥7 ✅（三场≥7） | 总分40分 ✅ | 无连续同分 ✅

**关键配角用法：**
- 护士长 — 敲打者。追上去喊「你老婆晕倒了」+ 沉默握我脉搏——这一个紧度替我问了「你嫁了个什么人」
- 我妈妈 — 代际对立面。7秒沉默——自己一辈子没被好好对待过，却劝女儿忍
- 老病人家属 — 信息传递者。唯一能传递「他把那句话对别人说了」这个信号的人

**可复用的结构：** 同院设定 + 三重配角 + 6场对抗1场收束。适用于「职场+家庭双战场」型冲突，两套场景共用一套配角群，切换成本低。

## 案例4：「知意花开」（2026-06-21通过，30分/33分调整版）

**核心特征：**

1. **追不回结局** — 不是"她回来了"，是"她走了，他没追上，各自安好"。和案例2形成镜像：案例2是"追回来了"，案例4是"追不回来"。两种结局都需要验证——追回来需要不可逆代价，追不回来需要女主真正放下。
2. **从向量数据库真实冲突提取** — 所有6场冲突均来自Qdrant因果推断库（人间theLivings非虚构），未自我发挥。每场标注参考来源名与烈度评分。
3. **开局价值观碰撞（6分→提至8分）** — 她说「你每天都这么忙」，他说「我在给这个家挣钱」。价值观对立：他认为「赚钱=爱」，她认为「看见=爱」。
4. **情绪曲线后段缓缓落下** — 冲突6从"害命8"（原生家庭阴影）改为"平和6"（各自安好），去掉撕心裂肺，改为云淡风轻。符合"前重后轻"原则：前期高烈度让读者疼，后期低烈度让读者愈合。
5. **多战场** — 医院（自杀/离婚）→ 花店（创业/拒绝）→ 酒局（醋意/质问）→ 花店门口（纠缠/决绝）→ 山区（支教/婚礼）→ 朋友圈（点赞/微笑）。
6. **配角三线** — 闺蜜（清醒的敲打者，"你早该离了"）、周牧野（温柔的救赎者，让她被看见）、心理医生（推动男主醒悟）。
7. **意象系统** — 日记本（她唯一的行李）→ 玫瑰刺（创业的血泪）→ 999朵玫瑰（他的挽回）→ "知意花开"（她的新生）。

**曲线（调整前33分版）：** 5→2→7→5→3→8（暗线）
**曲线（调整后30分版）：** 6→2→7→6→3→6

**硬门槛：** 前两场≥3 ✅（6+2） | 至少一场≥7 ✅（冲突3=7分） | 总分30分 ✅ | 无连续同分 ✅

**情绪曲线检查：** 前期需求不被满足（冷暴力→自杀）→中期觉醒（创业→拒绝）→后期被满足（各自安好）

**关键设计——冲突6的两次调整：**
- **第一版（害命8）**：插入女主母亲被冷暴力逼疯的暗线回忆。用户反馈："太黑暗，后面应该缓缓"
- **第二版（平和6）**：改为三年后朋友圈点赞微笑。情绪缓缓落下，余韵悠长。
- **教训**：追妻火葬场后期不需要再加原生家庭阴影来"补刀"——女主的觉醒已经足够，后期需要的是"愈合"不是"再撕开"。

**从数据库提取冲突的方法（已验证）：**
1. 用fastembed模型生成查询向量
2. 搜索Qdrant集合`causal_chains`，按语义相似度排序
3. 筛选烈度评分符合目标区间的冲突链
4. 提取"果-因-缘"结构，改编为小说场景
5. 标注参考来源名与烈度评分

**可复用的结构：** 开场暴击（自杀/离婚）→ 转折觉醒（创业/拒绝）→ 升级后悔（醋意/质问）→ 高潮决绝（纠缠/放下）→ 结局前成长（支教/婚礼）→ 尾声平和（点赞/微笑）。适用于"虐女→觉醒→追不回"型追妻火葬场。
