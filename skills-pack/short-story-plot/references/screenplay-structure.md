# 电影剧本结构参考（主角视角）

> 更新：增加了三幕结构、人物弧光、开局钩子、主题、结尾余韵。  
> 适用对象：短篇剧情（30,000字以内）——理论和电影一致，但节奏更紧凑。

## 零、故事的核心引擎

一个故事之所以让人放不下，是因为这五个东西在同步运转：

| 层 | 是什么 | 一句话 |
|----|--------|--------|
| **三幕** | 情节怎么安排——什么时候出什么事 | 开场→对抗→终局 |
| **弧光** | 主角怎么变——从什么人变成什么人 | 开头信什么→结尾信什么 |
| **主题** | 这个故事到底在说什么 | 人要走出自己的阴影，还是被它吞掉 |
| **钩子** | 第一页凭什么让人翻到第二页 | 一句话让人想知道"然后呢" |
| **余韵** | 翻完最后一页还留在心里的是什么 | 合上书之后的那口气 |

**这五个不是分开写的——是一个东西的五个面。**  
三幕是骨架，弧光是血肉，主题是心跳，钩子是门，余韵是关上门之后房间里的气味。

---

### 原则1：每场戏 = 主角想要一个具体的东西 + 有人挡路 + 结果（大概率没得到）

电影剧本里没有"纯描写""纯转场""纯内心戏"。每一场戏都必须回答三个问题：

1. **主角走进这场戏的时候想要什么？**——必须是具体的、看得见的
2. **谁不让他得到？**——必须有另一个有欲望的人在挡路
3. **结果是什么？**——通常是他没得到，或者得到了但付出了代价

**例（《教父》开场）：**
- 主角（殡仪馆老板）想要：柯里昂帮他的女儿报仇
- 谁挡路：柯里昂在听，但他没有答应——他说"你以前不尊重我，现在才来找我"
- 结果：没有直接答应，但留下了一个悬念——"总有一天我会找你帮个忙"

### 原则2：因果链用"Because/Therefore/But"推导

每场戏之间不是并列的，是因果的：

```
第一场戏发生。THEREFORE 第二场戏发生（因为第一场的结果）。
BUT 第二场出了意外（对手反击/命运介入）。THEREFORE 第三场戏发生。
```

**例（追妻火葬场）：**
```
第一场：她走了。
THEREFORE 他开始一个人生活——发现家里少了一个人是不一样的感觉。
BUT 他意识到自己不知道怎么开口——他从小被教育"爱是做出来的"。
THEREFORE 他去找老李喝酒——想问问别人是怎么过这一关的。
```

有了因果链，每场戏之间就不会断掉。没有因果链的故事就是"发生了一些事"。

### 原则3：主角必须主动——他不能只是"反应"

主角的欲望驱动每一场戏。他走进一场戏不是等事情发生，是**他想要一个东西**，所以他去做一件事。

| ❌ 被动写法 | ✅ 主动写法 |
|-----------|-----------|
| 她走后，他一个人在家很难过 | 她走后第七天，他打开鞋柜看到她的拖鞋——他把拖鞋拿出来放在了玄关 |
| 他想了想，觉得自己错了 | 他拨了老李的电话："你当年复婚的时候，是谁先找谁的？" |
| 他去了她的楼下，站了一会儿走了 | 他站在她楼下，发了一条消息。等了半小时，没回复。他又发了一条："我在楼下。" |

主动和被动不是看"他做了没有"，是看他**有没有一个想要的、具体的结果**。如果他去楼下只是为了"看看"，那是被动；如果他去楼下是为了"让她下来见我"，那是主动——即使她没下来。

### 原则4：对手也是主角——他也有他的欲望

好的对手不会觉得自己是坏人。他认为自己才是对的，而且他的"对"和你的"对"不能共存。

**写对手跟写主角一样的方法：**
- 他想要什么？（具体的、看得见的）
- 他为什么不让你得到？（不是因为他坏，是因为他的价值观要求他这么做）

**例（追妻火葬场）：**
她的价值观：**爱要能被看见，不是被安排。**
他不是对我不好——他对我很好。但他从来没有问过我"你想要什么"。他以为他给的就是我要的。我不想等他改——我不是在惩罚他，我是在选择我自己。

她不是坏人。她只是"两个人的频道不一样"这件事上选择了自己。

### 原则5：每场戏的主角目标必须是具体的有形物

| ❌ 抽象目标 | ✅ 具体目标 |
|-----------|-----------|
| 我想让她回来 | 我想让她回我消息 |
| 我想知道我做错了什么 | 我想看她朋友圈有没有删我的照片 |
| 我想道歉 | 我想让她答应见我一面 |

具体的目标才有"能不能达到"的悬念。抽象的目标没有尽头。

## 二、场景结构模板

每个场景的骨架：

```
[主角]想要[具体的东西]，
但[对手]做了[具体的事]挡路，
所以[主角]做了[具体的事]来回应，
结果[没得到/得到了但付出了代价/部分得到]。
```

**格式：**

```
场景[编号]：[一句话概括]
  我想要：[具体的东西]
  谁挡路：[角色]做了/说了什么
  我回应：[我的行动]
  结果：[现状——通常不是我想要的那个]
```

**例（追妻·第一场——她走的那天）：**

```
我想要：她不要走——至少今天不要
谁挡路：她已经叫好了车，行李在门口
我回应：我说"你先冷静一下"
结果：她看了我一眼，没有生气，拎着箱子走了——她不生气比生气更让我不安
```

**例（追妻·第七天——打开鞋柜）：**

```
我想要：找到一件她没带走的东西（作为"她还会回来"的证据）
谁挡路：鞋柜里她的拖鞋整整齐齐放着——但这不是"回来"的证据，这是"她收好了"的证据
我回应：我把拖鞋拿出来放在玄关——她回来就能穿到的位置
结果：放了三天，她没回来。拖鞋还在那里。我每天出门都看到它。
```

## 三、剧本式因果链和"因缘推演"的关系

因缘推演法的"第N天+因+缘+果"和剧本的"我想要+挡路+回应+结果"——本质是同一个结构：

| 因缘推演 | 剧本场景 |
|---------|---------|
| 我的因 → 我想要 | 我想要 |
| 缘（他的性格）→ 他做了什么 | 谁挡路 |
| 我对他的反应 → 我回应 | 我回应 |
| 果 | 结果 |

唯一的区别：**剧本式写法强制要求"我想要"是一个具体的、看得见的东西。** 因缘推演如果不加这个约束，容易写成"我在这里想了想"——没有欲望驱动，就不叫场景。

## 四、"但/因此"因果链检查

写完全部场景后，用"但/因此"连一遍：

```
场景1（她走的那天）:[内容]
  → 因此 场景2（第七天打开鞋柜）:我想找到"她会回来"的证据
    → 但 她没回来，拖鞋放了一周原封不动
      → 因此 场景3（老李请我吃饭）:我想找个人问问
        → 但 老李说"你是在等自己不丢脸吗"
          → 因此 场景4（电话我爸）:我想知道我父亲是怎么过来的
            → 但 我爸沉默
              → 因此 场景5（我去了她楼下）:我决定直接去找她
                → 但 她没有下来
                  → 因此 场景6（我发了那条消息）:我知道我要说什么了
```

如果某个场景连不上"因此"（比如"场景3发生了，但不是因为场景2的结果导致的"），说明因果链断了。要么删掉这个场景，要么重写前一个场景的结果。

## 五、"主角视角"检查清单

每个场景写完，用这张表自检：

- [ ] **我从这场戏走进来的时候想要什么？**——具体的东西
- [ ] **谁在阻止我？**——有名字的人，不是"命运""社会""公司"
- [ ] **我做了什么让他不舒服？**——不是"我做了什么"，是"我做了什么让他不舒服"
- [ ] **结果是什么？**——不是我想要的那个
- [ ] **结果是下一个场景的原因**——用"因此"能连上

## 六、三幕结构的分配（短篇版）

短篇30,000字以内，不能像电影那样分严格的page号，但比例是一样的：

```
第一幕「入场」：约前20%
  → 开场画面·主角的日常世界
  → 催化剂（一件打破日常的事）
  → 争论（主角犹豫要不要接招）
  → 跨过门槛（决定行动了）

第二幕「对抗」：中间约60%
  → 新世界试水（以为能搞定）
  → B故事（另一条线的伏笔）
  → 中点转折（以为要赢了/以为输了）
  → 坏人逼近（对手越压越紧）
  → 万念俱灰（觉得全完了）

第三幕「终局」：最后约20%
  → 最后一搏
  → 最终对决
  → 尘埃落定
  → 尾声
```

**比例分配表（以20,000字短篇为例）：**

| 幕 | 内容 | 字数 | 开始页 |
|----|------|------|--------|
| 第一幕 | 开场→催化剂→跨过门槛 | ~4,000字 | 1-12页 |
| 第二幕前段 | 试水→中点转折 | ~6,000字 | 13-30页 |
| 第二幕后段 | 坏人逼近→万念俱灰 | ~6,000字 | 31-48页 |
| 第三幕 | 最后一搏→终局→尾声 | ~4,000字 | 49-60页 |

**硬规则：**
- 第一幕必须在20%以内完成跨过门槛——读者在20%还不知道"这个故事要干嘛"就晚了
- 第二幕中点（50%位置）必须有一个明确的转折——让读者对后半段的预期完全不同
- 第三幕不要超过25%——收得太慢会泄气

**例（灵脉灭门案——分配）：**
```
20,000字短篇，约10个场景

第一幕（~4,000字，场景1-2）：
  场景1：陈家儿子在灵脉值守被宋家管事当众打了一顿（开场+催化剂）
  场景2：陈家去宗门告状，宗门说"按族规办"（争论+跨过门槛——他们只能靠自己了）

第二幕前段（~6,000字，场景3-5）：
  场景3：儿子经脉被废（以为能医好）
  场景4：宗门结论"灵脉反噬，非人为"（试水失败）
  场景5：陈父跪在执法堂外一天（中点——跪完了也没人出来）

第二幕后段（~6,000字，场景6-8）：
  场景6：儿子说"宋家管事进过我的值守段"（坏人逼近——发现了真相）
  场景7：陈父再次去找执法长老，被拒之门（万念俱灰——法律这条路堵死了）
  场景8：宋家管事的尸体被发现

第三幕（~4,000字，场景9-10）：
  场景9：陈父去执法堂自首，长老说"一命抵一命"（最后一搏+对决）
  场景10：陈父走出宗门——"我儿子的命，什么时候轮到官府定价了"（终局+尾声）
```

## 七、人物弧光——主角怎么变

**弧光的核心：主角开头信的"那件事"在结尾被推翻或印证了。**

不是所有的故事都需要主角变好。有些故事的主角不变，但读者看完了变了。有些故事的主角从好变坏。有些从头到尾没变——他证明了"我一开始就是对的"。

**弧光模型：**

```
阶段1：起点——主角信什么（通常是错的，或者是片面的）
  例：男主信"我对她好就够了，说不说爱不重要"

阶段2：催化剂——一件事逼他面对"那件事可能不对"
  例：她走了。她说"你从来没问过我想要什么"

阶段3：挣扎——他试图证明"我才是对的"——不断碰壁
  例：他买了花去她公司楼下等——她没下来。他觉得她任性

阶段4：迫近——不转变就来不及了
  例：她删了他所有联系方式

阶段5：终点——他信了什么（跟起点不同，或者更深了）
  例：他终于说出口了——"我一直以为我对你好就够了，但从来没问过你需要什么"
```

**弧光类型（选一种）：**

| 类型 | 起点 → 终点 | 适合什么故事 |
|------|------------|------------|
| **成长** | 天真→成熟 | 主角入世、历练 |
| **醒悟** | 坚信→怀疑→明白 | 追妻、复仇后被反噬 |
| **坚守** | 信→被考验→更信 | 英雄归来、信仰捍卫 |
| **堕落** | 纯良→被迫→沉沦 | 黑化、身不由己 |
| **颠覆** | 以为对→发现错→回不了头 | 发现自己才是施害者 |

**例（灵脉灭门版弧光）：**
```
类型：醒悟
起点：陈父信"宗门会给我一个公道"
中点：跪了一天没人出来——"宗门可能不会管"
终点："我儿子的命，什么时候轮到官府定价了"
——他不再相信任何秩序。他信了只能靠自己。
```

## 八、开局钩子——第一页让人翻到第二页

**钩子的本质不是"爆炸性开头"，是"制造一个必须在下一页解答的疑问"。**

### 四种钩子模式

**1. 问题钩：读者看到结果，想知道原因**
> "三个月前，他还是县城首富。现在他在菜市场卖菜。"  
> → 读者想知道：怎么走到这一步的？

**2. 谜题钩：读者看到异常，想知道解释**
> "她走的那天，没有吵架，没有摔东西。她把拖鞋整齐收进鞋柜，然后叫了车。"  
> → 读者想知道：为什么不吵架就走了？

**3. 情绪钩：读者被一句话击中，想知道这个人是谁**
> "我拉黑我爸那天，他说的最后一句话是'你跟你姐一样，都是疯子'。"  
> → 读者想知道：这句话为什么是一把刀？

**4. 预言钩：有人说了句关键的话，读者想知道它会不会成真**
> "我爸说，你断我香火，我死了怎么面对列祖列宗。"  
> — 三年后这句话在法庭上被念了出来。  
> → 读者想知道：这家人后来怎么样了？

### 禁止的开局
| ❌ 禁止 | 为什么 |
|---------|--------|
| "天很蓝，云很白" | 纯景物描写，没有任何疑问 |
| "他是一个……的人" | 人物介绍，不是故事 |
| 从出生开始写 | 短篇没有这个篇幅 |
| "事情要从那天说起" | 叙述者在偷懒 |
| 一段道理/感慨 | 读者还不认识人物，不在乎你的感慨 |

## 九、主题——这个故事到底在说什么

**主题不是"写一个复仇的故事"——那是类型。  
主题是"这个故事看完，读者心里留下的那句话"。**

### 怎么定主题

在动笔之前，用一句话回答：

> **如果这个故事只剩一句话，那句话是什么？**

**例：**
- 灵脉灭门案 → "穷人的公道不在法律那里"
- 追妻火葬场 → "爱不能被安排好，它要被看见"
- 冷暴力30年 → "比恨更狠的是漠视"
- 家暴一生 → "婚姻两个字可以绑一个女人一辈子"

### 主题怎么发挥作用

**主题在故事里存在三次：**

1. **开头被陈述（但没人当真）**  
   陈父说"宗门会给个公道的"——读者知道他在赌，但不知道最后能不能赌赢

2. **中间被反复检验**  
   每一次去执法堂、每一次被告知"没有证据"，都在检验"到底有没有公道"

3. **结尾被印证或推翻**  
   他用自己的方式讨了公道——"我儿子的命，什么时候轮到官府定价了"  
   → 主题被印证了：没有公道，只有自己讨

**没有主题的故事像什么？**  
像一个人说了很多话，但你不知道他到底想说什么。冲突再精彩，看完也就忘了。

## 十、结尾余韵——合上书之后

**一个好的结尾不是"写完了"，是让读者翻完最后一页后，坐在那里多待了几秒钟。**

### 四种收法

**1. 印证式结尾**
> 最后这句证明了开头那句话是对的。
> 例：陈父走出宗门，"我儿子的命，什么时候轮到官府定价了"
> → 印证了：开头他以为宗门有公道，结尾他知道没有。

**2. 回旋式结尾**
> 结尾和开头呼应——同一句话、同一个动作、同一个地方，但意义完全变了。
> 例：开头是他站在她楼下等，结尾是他又站在她楼下——但这次他不是在等。
> **适合：** 追妻、救赎、时间跨度大的故事。

**3. 留白式结尾**
> 没有把话说死。读者知道故事完了，但不知道结局具体是什么。
> 例：她透过窗户看见楼下有人。「他走到路灯下站了一会儿，然后走了。」
> → 她会不会下来？不知道。但读者心里有自己的答案。
> **适合：** 开放式结局、有后劲的短篇。

**4. 沉默式结尾**
> 什么都没发生。但什么都没发生才是最有力量的那个结果。
> 例：冷暴力30年——母亲到死都没说那句话。
> → 最狠的不是她说了什么，是她什么都没说。
> **适合：** 悲剧、真实事件改编。

### 收尾检查表
- [ ] **最后一句有价值吗？**——不是"他睡了""天亮了"这种废话
- [ ] **读者合上书后，会在心里再待三秒钟吗？**
- [ ] **结尾跟开头有关系吗？**——同一个地方/同一个人/同一句话/同一个问题？
- [ ] **主题在这轮被印证/撼动/推翻了吗？**

## 十一、五大引擎串联实例（灵脉灭门案）

```
钩子：陈家儿子在灵脉值守时被宋家管事当众打了一顿

↓

三幕：
  第一幕（20%）：被打→告状→宗门不管
  第二幕（60%）：儿子被废→查不出真相→跪求无门→复仇
```
